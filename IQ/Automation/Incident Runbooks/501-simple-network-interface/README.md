# Simple Example Network Interface Incident

Runbook for network-interface incidents. Triggered by a network_interface event, it queries interface status or traffic data (depending on the primary indicator) and renders UI visualizations—no remediation steps.

After downloading the [runbook file](./Simple%20Example%20Network%20Interface%20Incident.txt), go to the IQ **Automation** page and import it under **Incident Runbooks**.
